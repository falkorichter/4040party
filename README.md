# 4040party

HTML Landing Page for the 4040 Party

## Overview

This is a beautiful, responsive HTML landing page for the 4040 Party event. The page includes sections for:
- Party poster display
- Event details (date, time, location, dress code)
- Additional information
- RSVP section

## Setup Instructions

1. **Add Your Poster Image**
   - Save your party poster as `poster.jpg` in the same folder as `index.html`
   - Supported formats: JPG, PNG, GIF
   - For best results, use a high-quality image

2. **Customize Event Details**
   - Open `index.html` in a text editor
   - Replace the placeholder text in brackets `[Add ...]` with your actual party information:
     - Date
     - Time
     - Venue name and address
     - Dress code
     - RSVP deadline
     - Contact email

3. **Update RSVP Link**
   - Find the line with `mailto:your-email@example.com`
   - Replace with your actual email address or link to an RSVP form

## Viewing the Page

Simply open `index.html` in any web browser:
- Double-click the file, or
- Right-click and select "Open with" your preferred browser
